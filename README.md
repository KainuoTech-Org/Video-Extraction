# 视频导出工具 (Video Export Tool)

这是一个基于 Python 的简单的视频链接解析和导出工具，旨在帮助用户方便地下载自己喜欢的视频内容进行离线观看。

## 功能支持

- **Bilibili**: 支持解析 BV 号及 b23.tv 短链接。
- **全民K歌 (KG)**: 支持解析分享链接。

## 部署

本项目配置了 `vercel.json`，可以直接部署在 Vercel 上使用。

### 本地运行

1. 安装依赖:
   ```bash
   pip install -r requirements.txt
   ```

2. 运行应用:
   ```bash
   python main.py
   ```
